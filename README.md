SisGESC - Sistema de Gestão Escolar

Projeto de Modelagem de Banco de Dados desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas da Universidade Cidade de São Paulo (Unicid).

 Sobre o Projeto

O SisGESC é um banco de dados relacional para uma escola privada de Ensino Fundamental. O sistema integra três módulos principais:

- **Módulo Acadêmico**: Gerenciamento de alunos, cursos, disciplinas, turmas, matrículas, notas e faltas.
- **Módulo Financeiro**: Controle de contratos, mensalidades, pagamentos, contas a receber e inadimplência.
- **Módulo RH**: Cadastro de funcionários, professores, vínculos com disciplinas e carga horária docente.

O banco foi modelado seguindo as boas práticas de projeto relacional, com normalização até a 3ª Forma Normal (3FN).

 Tecnologias Utilizadas

- Modelo Entidade-Relacionamento (DER)
- Normalização de dados
- Constraints (NOT NULL, UNIQUE, DEFAULT, CHECK)
